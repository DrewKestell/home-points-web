HomePoints

HomePoints is a utility used to filter GPS data. It started as a console application written in Python, but recently I recreated it as a web application built with angular.js. You can define any number of circles, and any GPS data that falls outside of those circles will be deleted. Read More on Github.

Instructions

  • First, click either the Single or Multiple tab depending on whether you're working with one initial radius or multiple.
  • If you have one initial location, enter the latitude/longitude and radius in the input fields to the left. If you have multiple initial locations, list them in a .csv file and upload that file to the left. Click here to see an example of how your file should be formatted.
  • Upload your raw GPS data in a .csv file. Click here to see an example of how your file should be formatted.
  • Click Submit. Processing your data may take some time depending on the amount of GPS data being cleansed. Once processing is complete, a new .csv file will start downloading in your browser.
